OBS: Dette er kun til brug på windows, hvis du har brug for andet skal du kigge på originalen.

1. installer java i mappen c:\jre
2. Du skal bruge nogle oplysninger som står i viewer.jnlp filen.
	Download filen ved at åbne konsollen via idrac web ui.
	Åben viewer.jnlp med en tekst editor.
	Kopier url for "avctKVM.jar" og download filen "avctKVM.jar" (normalt er den https://IP:443/software/avctKVM.jar)
	Flyt "avctKVM.jar" til c:\jre 
3. Udfyld start_quick.bat med dine idrac oplysninger.
4. Kør start_quick.bat

Opret en sag hvis du har problemer.
